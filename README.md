# Advanced Spatial Analyses (ASA) – Coursework

This repository collects my work for the **Advanced Spatial Analyses (ASA)** course in the Earth Sciences MSc at Vrije Universiteit Amsterdam.

The course covers Python fundamentals and spatial analysis. This repo shows how I used code to work with environmental and spatial data.
