# ASS 2.3 – Wildfire Simulation and Prevention

This assignment develops a dynamic simulation model to understand how wildfires spread over time. A grid-based system is used to represent a forest, where each cell tracks wood availability, temperature, and burning state.

## Code structure

Main workflow:

- Initialise a grid representing forest conditions  
- Define rules for fire ignition, heat transfer, and fuel consumption  
- Simulate fire spread across time steps  
- Model heat loss to the atmosphere and neighbouring cells  
- Implement ignition and extinction thresholds  
- Test interventions such as firebreaks  

## Skills and tools demonstrated

- Building dynamic models in Python  
- Working with NumPy arrays for grid-based simulation  
- Implementing iterative processes and time steps  
- Modelling environmental processes using rule-based systems  
- Visualising simulation outputs
