# ASS 2.4 – Flood Delineation

This assignment applies machine learning to classify flooded and non-flooded areas using satellite imagery. The analysis is based on Sentinel-2 RGB data and evaluates different classification approaches.

## Code structure

Main workflow:

- Extract RGB values from satellite imagery  
- Prepare labelled training data for flooded and non-flooded points  
- Train machine learning models (Random Forest, Logistic Regression, KNN)  
- Evaluate model performance using accuracy, precision, recall, and F1-score  
- Compare results across models  
- Test model generalisation on a different dataset  

## Skills and tools demonstrated

- Supervised machine learning in Python  
- Model evaluation using classification metrics  
- Working with remote sensing data  
- Data preprocessing and feature extraction  
- Comparing model performance and interpreting results
